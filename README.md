# Vue-Typescript - 3 Steps Only!

<div align=center>
    <img src="public/vue-typescript.png" width="250"/>
</div>

### Project setup :
```
npm install -g @vue/cli
```

### Install template loader, Typescript can stream at webpack

```
npm install --save vue-template-loader webpack-stream
```

### Start Serve

```
npm run serve
```
